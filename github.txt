при создании папки на гитхабе (репозиторий), выбираем приват или общий доступ
добавить файл описание readme.md
добавить файл описание gitignore.txt
git remote add origin https://github.com

git push -u origin main - добавить все изменения в етку 
git branch -M main - изменение имени ветки
