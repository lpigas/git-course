git init - инициализация изменяемых файлов
git status - статус измененных файлов
git add <file>- добавление изменений 1 файла
git add . - добавление изменений всех файлов в папке
при добавлении гит игнор описываем файли или полностью файлы которые не будут передаваться в гит
$ git branch <name vetki> - создание  ветки 
$ git branch -D <name vetki> - удаление ветки 
git checkout <имя> - смена ветки
git checkout -b name - переход в новую ветку с созданием ее
git merge lpigas - добавление ветки в основную
